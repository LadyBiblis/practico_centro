Importantes HTML tags

Esta página está diseñada como una rápida referencia de los más comunes tag desarrollados en el curso, basado en Learn Enough HTML to Be Dangerous. En el proceso de realizarlo, vamos a aprender como hacer tablas HTML, via table y otros tags relacionados.

Las tablas debajo no incluyen todos los HTML tags, pero listan muchos de los más importantes

Block elements

Tag Nombre Propósito
h1–h6 headings incluye un encabezado (levels 1–6)
p paragraph incluye un párrafo de texto
table table incluye una tabla
tr table row incluye una fila de datos
th table header hace un encabezado de tabla
td table data incluye una celda de datos
header header unidad lógica de encabezado
div division división imaginaria de página sin impacto en la apariencia
ol Ordered list Lista ordenada por números o letras
ul Unordered list Lista desordenada
li List item Items que van dentro de una lista, sea ordenada o desordenada

Inline Elements

Tag Nombre Propósito Ejemplo Resultado
em emphasized enfatizar el texto <em>technical sophistication</em> technical sophistication
strong strong texto en negrita <strong>al menos un millón de personas</strong> al menos un millón de personas
a anchor hacer un enlace <a href="http://learnenough.com/">Learn Enough</a> Learn Enough
img image incluye una imagen <img src="http://bit.ly/2sj60X6" alt="juampa villa">
code code escribe código fuente <code>hola mundo</code> hola mundo
span span <span>secciona una porción de la línea</span> secciona una porción de la línea sin apariencia en la página secciona una porción de la línea
